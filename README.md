## Using Examples

Add the following to your html code.
First, import the following files:

```html
  <div class="search">
    <i class="fas fa-search"></i>
  </div>
  <div class="heart">
    <i class="fas fa-heart"></i>
  </div>
  <div class="shoping">
    <i class="fas fa-shopping-cart"></i>
  </div>
```

Second, import the following files

